7 Days To Die Public Server (PVP)

| IP: 7days.vaido.world

Server Information

The server is based in Europe/Lithuania with a focus to be as Open as possible to the community.
If you play on this server, you can download the map and use it on your own server.

If you want to suggest a modlet for the server, you can always suggest it via merge request
or participate in other discussions in the Game Server settings Github Repository.

Suggestions to change the Game Server Settings are also welcome.

Since this project is free and open. It would be great to have support via donations.

Server Settings

  • server mods folder
  • server map folder
